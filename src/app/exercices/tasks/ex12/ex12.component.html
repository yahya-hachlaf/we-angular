Aufgabe 1.2. HTTP

HTTP-Statuscode 200 OK gibt an, dass eine Anfrage erfolgreich verlaufen ist. HTTP Status Code 301 bedeutet, dass der URI der angeforderten Ressource geändert wurde. HTTP Status Code 400 bedeutet, dass der Server die Anfrage aufgrund einer ungültigen Syntax nicht verstehen konnte. Der Client hat keine Zugriffsrechte auf den Inhalt. Die Abfrage belebig oft stellen um die gleiche Ergebnisse zu bekommen oder Man kann auch die Access-Control-Max-Age benutzen,somit maximale Anzahl von Sekunden, für die die Ergebnisse zwischengespeichert werden können.
